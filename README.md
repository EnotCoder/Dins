# Dins

<a align="center">This console program will help you find</a></p>
<a align="center">the files you need in folders.</a></p>
<a align="center">Just enter "dins name file"</a></p>

<h1 align="center">DOWNLOAD</h1>
<h3 align="center">		cd "folder with dins"</h3>
<h3 align="center">		sudo cp dins /usr/local/bin/</h3>
<h1 align="center">DELETE</h1>
<h3 align="center">		sudo rm /usr/local/bin/dins</h3>
